# Task 3  AI-Driven Development - 30-Days Challenge  

## PART A Research Questions

**1. What new improvements were introduced in Gemini 3.0?**  
Gemini 3.0 introduces advanced reasoning, improved multimodal understanding, agentic coding, interactive visual layouts and generative interfaces. It provides more intelligent, concise and context-aware responses.  

**2. How does Gemini 3.0 improve coding & automation workflows?**  
Gemini 3 Pro supports agentic coding, allowing users to create complex code, multi-step tasks and automated workflows with a single prompt. Integration with CLI and Google AI Studio makes coding faster and more precise.  

**3. How does Gemini 3.0 improve multimodal understanding?**  
Gemini 3.0 can understand text, images, and videos. It can generate apps, interactive UI and simulations from visual inputs. Document understanding, spatial reasoning and video comprehension have also improved.  

**4. Name any two developer tools introduced with Gemini 3.0.**  
- **Google AI Studio**  Enables building apps and interactive projects with a single prompt.  
- **Gemini CLI**  Allows executing terminal commands, coding and automation efficiently.
